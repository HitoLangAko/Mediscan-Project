# MediScan Demo Script

**Pitch:** MediScan is an offline-first medicine scanner and personal medicine vault. It uses QVAC-ready local extraction to read medicine labels, then matches the extracted data with a local reference database and classifies each scan as Verified, Needs Verification, or Expired.

## Demo Steps

1. Show the empty vault.
2. Open Scan and choose Verified label demo.
3. Explain that QVAC extracts medicine name, strength, barcode, and expiration date.
4. Show Verified result and save it.
5. Scan the expired sample and show Expired status.
6. Scan the pill-only sample and show Needs Verification.
7. Explain that pill appearance alone is unsafe because look-alike pills can contain different ingredients.
8. Open Database and search for a medicine.

## Closing

MediScan does not replace doctors or pharmacists. It helps families organize medicines safely and privately, especially in offline or low-connectivity settings.
